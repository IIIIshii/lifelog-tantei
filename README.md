## MVP(最小限の機能)の定義

### 1：聞き込み(AIとの対話)
・AIが質問し、その答えに対して1回~2回深堀りする。

### 2：証拠提出(入力)
・ユーザーが選択肢をタップか、短い選択肢で答える。

### 3：事件簿生成(日記生成)
・会話の内容を基に、AIが100~300字の日記を自動作成する。

### 4：証拠保存(DB連携)
・作成した日記をFirebase Firestoreに保存する。

### 5：事件簿の閲覧(履歴)
・保存された過去の日記をリスト形式で表示し、読み返せるようにする。
