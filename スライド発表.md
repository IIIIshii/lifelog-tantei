# ライフログ探偵

これは、自分で文章を書かなくても、質問に答えていくだけで日記を作成できるアプリです。

# ざっくり説明

決めた時間になるとリマインドが飛んでくる
アプリを開くといろんな質問が飛んできてその日のことを聞き出す
睡眠時間とか 自分で記録したいものをカスタマイズしておくこともできる
回答をもとに日記テキストを生成
睡眠時間など 行動習慣を表やグラフで確認できる


# なんで作ったか
日記あるある 続かない
なんで？ → 忘れてた、文字に起こすのが面倒、時間がない
一番あるのは[能動的に動くのがダルい！]
従来は[日記は能動的に書くもの]
質問に[受動的に]答えていくだけで日記がかける

# こだわって実装した内容
ユーザーにできるだけ頭を使わせない、楽をさせる 
    ・できるだけ選択肢式の質問を増やす！ [テキストうつのだるいよね？]
    ・思い出しアシスト機能
        ・今朝何した、午後は何した、とか答えていくうちに一日を思い出してく
            →書きたい出来事が定まる
                それについて深堀り
＆
書きたいことがあるなら邪魔せず書けるようにする
    ・好きに書き加えたり編集したりできる
    ・拡張性のために一応会話ログを全部残す
